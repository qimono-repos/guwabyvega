## 3. UI Guidelines & D-Pad Focus Primitives
This application is designed specifically for TV screens (10ft interface / living room viewing), meaning:
- **No touch layout**: User navigates exclusively with a D-pad remote or keyboard arrow keys.
- **Focus Indicators**: Focused items must be clearly styled (transform scale, distinct background colors e.g. blue `#0074B8` for default and orange `#FF6200` for focused).
- **React Native TV Primitives**:
  - Always use `TVFocusGuideView` (imported from `@amazon-devices/react-native-kepler` SDK) to wrap focusable grid components.
  - Use `hasTVPreferredFocus` to mark the default/preferred starting element (typically the `home` tile).
  - Use `accessibilityRole="button"`, `accessibilityLabel`, and `testID` to ensure reliable user navigation, screen-reader support, and test suite execution.
- **Visuals**: Maintain large font sizes, clear labels, high contrast, and dynamic transition styling (such as micro-animations like the scale transform `[{scale: 1.1}]`).
